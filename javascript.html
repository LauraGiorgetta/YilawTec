Estructura Básica

HTML, Lenguaje de Etiquetas

Es un lenguaje de maquetación, en el cual se utilizar etiquetas para darle la estructura a la página.

<etiqueta></etiqueta>

Todo lo que está entre < y > son etiquetas → <etiqueta>

Toda etiqueta que se abre, debe cerrarse, el cual se cierra con la misma etiqueta agregándole una / después del primer < → <etiqueta> </etiqueta>

Usualmente este formato se utiliza para escribir texto, más etiquetas, o funciones dentro de esas mismas etiquetas

<etiqueta>Codigo, escrituras, etc...</etiqueta>

Aunque hay Etiquetas que no pueden contener nada dentro, que son las etiquetas abiertas, a lo largo que vayamos aprendiendo vamos a descubrir cuales son esas etiquetas.
